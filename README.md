# Cli tool (vtt)
## How to use ?
- first navigate to your working directory and initialize the tracking workspace `vtt init`
- second setup your `.vttignore` file similar to your `.gitignore`
- third try your `vtt status` command
- fourth add things to track via `vtt track [glob pattern]`
- fifth use the tasks command tree , run `vtt task`


# Note :
this is just a simple version with a basic Tasks tracking system , no change log generation and no logging added yet!
